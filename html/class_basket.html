Detailed Description

Klasa tworząca tzw. "koszyk zakupów".

Tworzy "koszyk", w którym umieszczać można wybrane komponenty. Klasa stworzona wg wzorca Singleton. Posiada kontener typu "vector", w którym przechowywane są dodane przez użytkownika komponenty, a także kontener typu "list" przechowujący obiekty typu QString (QStringList - specjalny kontener Qt).
